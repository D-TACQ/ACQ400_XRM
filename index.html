ACQ400_XRM README

xrmIoc creates the custom functionality and public interface for the XRM series devices.

The XRM functionality is created in a modular fashion from sub-classes of acq400_asynPortDriver, a subclass of asynPortDriver.

export XRM_INST2="STRATEGY={STR|SPY};REDIS_HOST={a.b.c.d};REDIS_PORT=nnn;REDIS_MKEY=string"
IN1STR_cmd=command for IN1STR
IN2STR_cmd=command for IN2STR
